# Task7- (Laundry Service Responsive Issues)

## Project Description
A responsive Laundry Service Landing Page built using HTML and CSS (Flexbox).
This project demonstrates modern layout techniques and responsive design using media queries.

## Features
- Logo on left, menu in center, username on right
- Flexbox-based layout (no inline/block hacks)
- Hero section with text + image
- Book a Service button
- Fully responsive (Desktop, Tablet, Mobile)

## Responsive Design

Desktop View
- Horizontal layout using Flexbox
- Menu items visible in navbar
- Hero section split into two columns

Tablet View (≤ 992px)
- Reduced font sizes
- Smaller image size
- Maintains row layout

Mobile View (≤ 600px)
- Navigation links hidden
- Only logo and username visible
- Hero section changes to column layout
- Center-aligned content

## Technologies Used
- HTML5
- CSS3
- Flexbox
- Media Queries

##  How to run it 
1. Download or clone the repository.
2. Open the project folder.
3. Double click **index.html** or open it in any web browser.
